== Description ==

O Canal Piloto é um site de entretenimento para aviação e formação aeronáutica criado em 2010. Mantendo o formato de blog, o site atualmente possui artigos em texto, podcast, sessão de dúvidas, imagens de entretenimento, canal de conteúdo original e canal de conteúdo colaborativo. A principal base para o site se manter é unir entretenimento e informação para seu público alvo, deste modo o “CP” é acessado diariamente tanto por aqueles que procuram detalhes sobre a formação aeronáutica, quanto por aqueles que desejam apenas passar o tempo consumindo conteúdo produzido por, e para, aviadores. 

http://canalpiloto.com.br/sobre/

This plugin creates a widget that displays a list of the last 5 episodes of CPCAST.


http://www.canalpiloto.com.br

If you like the CPCAST and has a website in WordPress download and activate this plugin. =)
